# 🐾 Wildlife Tracking System

## 📌 Overview
This project is designed to track wildlife animals using GPS and GSM technology. It helps researchers and forest departments monitor animal movement and ensure safety.

## 🎯 Objective
To provide a low-cost and efficient solution for tracking animals in real-time.

## ⚙️ Components Used
- Arduino Uno
- GSM Module (SIM800L)
- Power Supply
- Antenna

## 🔧 Working
The GPS module collects real-time location data (latitude & longitude).  
This data is sent via GSM module as an SMS to a registered mobile number.

## 🚀 Features
- Real-time location tracking
- SMS alerts
- Low-cost system
- Useful for wildlife conservation

## 🔮 Future Improvements
- Mobile app integration
- Geo-fencing alerts
- AI-based animal behavior prediction
